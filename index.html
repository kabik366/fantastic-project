HTML - структура веб-страницы

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц. Он определяет структуру и содержание веб-страницы.

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая веб-страница.</p>
</body>
</html>

Основные HTML-теги

  • <h1>-<h6> - заголовки разного уровня
  • <p> - параграф текста
  • <a> - ссылка
  • <img> - изображение
  • <ul>, <ol>, <li> - списки
  • <div> - блочный контейнер
  • <span> - строчный контейнер

Совет: Всегда используйте семантические теги (например, <header>, <nav>, <main>, <footer>) для лучшей структуры и доступности вашего кода.

CSS - стилизация веб-страницы

CSS (Cascading Style Sheets) - это язык стилей, который используется для описания внешнего вида документа, написанного на HTML.

/* Пример CSS */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    color: #333;
    background-color: #f4f4f4;
}

.container {
    width: 80%;
    margin: auto;
    padding: 20px;
}

.button {
    display: inline-block;
    background: #4CAF50;
    color: white;
    padding: 10px 20px;
    text-decoration: none;
    border-radius: 5px;
}

Основные концепции CSS

  • Селекторы - способы выбора элементов для стилизации
  • Свойства и значения - определяют, как должен выглядеть элемент
  • Блочная модель - margin, border, padding, content
  • Flexbox и Grid - современные методы макета
  • Адаптивный дизайн - медиазапросы для разных устройств

JavaScript - интерактивность на странице

JavaScript - это язык программирования, который позволяет вам создавать динамически обновляемый контент, управлять мультимедиа, анимировать изображения и многое другое.

// Пример JavaScript
document.addEventListener('DOMContentLoaded', function() {
    // Код выполнится после загрузки DOM
    const button = document.getElementById('myButton');
    const output = document.getElementById('output');
    
    button.addEventListener('click', function() {
        output.textContent = 'Кнопка была нажата!';
    });
});

Основы JavaScript

  • Переменные - let, const, var
  • Типы данных - числа, строки, объекты, массивы и т.д.
  • Условные операторы - if/else, switch
  • Циклы - for, while, do...while
  • Функции - объявление и вызов
  • DOM-манипуляции - работа с элементами страницы
  • События - обработка действий пользователя
